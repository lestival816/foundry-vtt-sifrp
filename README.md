# A Song of Ice and Fire Roleplaying: Game of Thrones Edition (Unofficial)

This is an unofficial implementation for **A Song of Ice and Fire Roleplaying: Game of Thrones Edition** by **Green Ronin Publishing**. This system code is based on the Boilerplate System by Asacolips.

## Features

- Character sheet support for ASOIAF RPG (SIFRP) core rules (playbale character and non playable character sheets)
- Combat tracker with SIFRP-specific combar initiative
- Dice rolls with custom modifiers 

## Usage

1. Create a new World
2. Select **A Song of Ice and Fire Roleplaying** as the system
3. Add actors, items, and scenes
4. Enjoy Westeros in Foundry VTT!
